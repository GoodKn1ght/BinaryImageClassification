Task is to detect artefacts on image such as tattoos or some writings on it. 
Dataset is imbalanced (90 / 10)
Tried multiple techniques: threshhold, oversampling, punishing for ignoring smaller class
Tried different models: basic CNN, ResNet, Pretrained EfficientNet
dataset - https://www.kaggle.com/datasets/goodkn1ght/artefact-detection/data

notebook - https://www.kaggle.com/code/goodkn1ght/artefact-task
